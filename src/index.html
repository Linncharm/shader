🎨 Shader 学习之旅

📚 学习路线 (基于Three.js)

1. 颜色渐变 - Fragment Shader基础
2. 时间动画 - 动态uniform传值
3. 图形绘制 - SDF距离场函数 (阶段2和3已完成)
4. 纹理贴图 - Three.js纹理系统
5. 几何变换 - Vertex Shader变换
6. 光照效果 - 内置光照系统扩展
7. 高级效果 - 后处理通道、粒子着色器

Demo 1: 颜色渐变 (Three.js版本)

使用Three.js的ShaderMaterial创建自定义shader。Three.js简化了WebGL操作，让我们专注于shader逻辑。
• 使用gl_FragCoord获取片段坐标
• 通过uniform传递分辨率参数
• Three.js自动处理顶点变换

Demo 2: 时间动画

学习如何在shader中使用时间参数创建动态效果。通过performance.now()更新uniform实现实时动画。
• 时间驱动的波浪图案
• 颜色在多种色彩间循环变化
• sin/cos函数创建周期性动画

Demo 3: SDF图形绘制

学习距离场(SDF)技术绘制复杂图形。SDF是现代shader中创建图形的强大工具。
• 圆形、矩形、圆角矩形的SDF函数
• 图形变换：旋转、缩放、位移
• 平滑混合多个图形
• 边缘光晕和高光效果

Demo 4: 波浪效果

深入学习正弦波动画，创建复杂的波浪图案。展示多种波浪的叠加和干涉效果。
• 同心圆波浪从中心扩散
• 水平、垂直、对角线波浪
• 双点源干涉图案
• 波峰高光和深度感渲染

Demo 5: 高级SDF技术

掌握SDF的高级应用：复杂图形、布尔运算、空间变换。展示SDF在复杂图形设计中的强大能力。
• 六边形、星形等复杂SDF函数
• 布尔运算：并集、差集、交集
• 空间变换：扭曲、重复、旋转
• 平滑运算和高级混合技术
